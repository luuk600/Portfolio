Expense-Calculator

This was the last project I've made for my first year.

Mostly when I pick my projects I think of something that I can use instead of what's nice to have on my resume.

I have used this to calculate my expenses and income since i'm turning 18 years old.

And when you turn 18 you need to pay for a lot more stuff.

This project has been made with HTML, CSS and javascript.

I think that the hardest thing about this project was the javascript because it was pretty new and i didn't know everything.

I've learned a lot from this project like what DOMContentLoaded means which I now have used in more projects.

and the storing of the value's was also pretty challenging, but it was done in time.
