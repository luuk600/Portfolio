Shine

This was my project for shine.

Shine basically is an expo for a big project that we make to show to our parents and attract new students.

I've made this in HTML and CSS in Visual Studio Code it took me about 4 hours to make and was not that.

I always start with making the NavBar because I think this is one of the most important parts of the project.

Because if you have a web shop, people can't buy something if they can't even get to it.

The hardest part about this project was the SINTLUCAS logo because the website is black but the logo is black too.

So I had a few problems with finding out where the logo was until I just copied the fond and made it text in Photoshop.

Since we hadn't learned how to use a grid layout, the code is a mess since we learned grid making websites like this became easier.

It was a fun to make project but wasn't really challenging, so I hope that next time it will be more interesting

But still, I learned something and that's really the most important part
